# 25-26-Strukdat-Tubes19_MLL01
Tugas Besar Struktur Data, Desember 2025

Membuat console app untuk manajemen data supplier dan produk dengan struktur data MLL (Multi Linked List).

List parent: Supplier dengan SLL (Singly Linked List)

List child: Produk dengan DLL (Doubly Linked List)


Fungsionalitas:

a. Menambahkan supplier. 

b. Menambahkan produk. 

c. Mencari supplier tertentu. 

d. Menghubungkan produk ke supplier. 

e. Menampilkan produk berdasarkan supplier. 

f. Menghapus supplier beserta produk. 

g. Menghapus produk dari supplier. 

h. Menampilkan seluruh supplier dan produk. 

i. Menghitung jumlah produk per supplier. 

j. Menampilkan supplier dengan produk terbanyak.
